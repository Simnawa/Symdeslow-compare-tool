# Symdeslow Compare Tool

A Google Sheets-based comparison tool that highlights differences in four colors and generates detailed reports. Ideal for analysts, auditors, finance teams, administrative staff, warehouse, sales, procurement, production planning teams, IT, quality control, supply chain teams, operations managers, researchers, and anyone working with routine or more complex datasets.

## Key Features

- Perform direct column-to-column and key-based comparisons
- Highlight differences using four distinct colors for easy visual tracking
- Automatically generate difference and duplicate key reports
- Suitable for routine or complex datasets across multiple departments and teams

## How to Use

1. Make a copy of Symdeslow Compare Tool through the link received.
2. Paste your data in Sheet 1 and Sheet 2.
3. For key-based comparison: Set your comparison key and data range for comparison. 
   For direct column-to-column comparison: skip this step.
4. Click the "Compare Now" button to generate highlights and reports.
